| Symbol    | Meaning                | Comments                        |
|-----------|------------------------|---------------------------------|
| tikey     | App Key                | App, API, CSS, Decal. etc.      |
| wskey     | Workspace Key          | Location on LAN                 |
| fskey     | Workspace Key          | Location on File System         |
